# Проект: Место

## Цели проекта
- Сделать сайт адаптивным;
- Реализовать:
* попап редактирования профиля / открытие и закрытие попапа.
* 6 карточек создаются при помощи JS;
* при создании карточек мест должен использоваться template , описанный в index.html ;
* форму добавления карточки, которая будет правильно свёрстана, открыватьс и добавлять карточку;
* нажатие на кнопку лайка;
* удаление карточки;
* плавное открытие и закрытие модального окна CSS-стилями.


## Используемые технологии:
- Методология БЭМ;
- Фйловая структура Nested БЭМ;
- @media;
- Flexbox;
- DOM.
